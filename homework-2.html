Homework 2: Distributions and the Caesar Cipher

This assignment is organized in two parts. Part A focuses on datasets and distributions (with MS Access for data creation), while Part B uses JavaScript to analyze text and implement Caesar cipher operations.

Part A — Dataset and Distribution

Concepts

Dataset is a collection of observations organized in rows (records) and columns (variables). Each variable represents a measurable attribute (e.g., Age, Department, LoginFailures), and each record represents one observation (e.g., a user, a session, a host).

Distribution describes how the values of a variable are spread. A univariate distribution focuses on one variable at a time (e.g., frequency of Departments). A bivariate distribution (cross-tab) examines the joint frequency of two variables (e.g., Department vs. MalwareDetected).

Workflow (using MS Access)

  1. Create a new table (e.g., “UserSecurity”). Suggested fields: Department (Short Text), Age (Number), LoginFailures (Number), MalwareDetected (Yes/No).
  2. Insert 15–30 rows of data (realistic values).
  3. Export the table as CSV for analysis on this page (External Data → Export → Text File).
  4. Compute up to k ≤ 3 univariate distributions (frequency tables) and optionally a bivariate (cross-tab) using the tools below.
